## 프로젝트 소개
모바일 프로젝트 및 최적화 경험을 목적으로 진행 하였습니다.

방대한 적 웨이브와 성장 시스템을 갖춘 캐주얼 2D 모바일 뱀서라이크류 게임입니다.

레벨업 시 무기나 악세서리를 선택해 캐릭터를 강화하며, 특정 무기 레벨에 도달하면 스킬이 해금되어 전략적인 전투와 재미를 더했습니다.
